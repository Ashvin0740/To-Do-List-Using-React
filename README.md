# To-Do-List-Using_React

### simple todos App

## How to get it in your PC&Laptop
```
git clone https://github.com/Ashvin0740/To-Do-List-Using-React.git
cd react-chatbot
npm install
npm start
The demo will be live at http://localhost:3000
```
